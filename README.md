# Staff Leave Management System (SLMS)

A web-based Staff Leave Management System developed using Python and Django to automate leave applications, approvals, and leave record management. The system helps organizations reduce paperwork, improve communication, and manage staff leave requests efficiently through a secure and user-friendly platform.

## Features

* Staff Registration & Login
* Leave Application System
* Admin Approval/Rejection
* Leave Status Tracking
* Secure Authentication
* Responsive Dashboard
* Database Management
* Real-time Leave Updates
